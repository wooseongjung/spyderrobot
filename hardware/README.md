# hardware/

All physical-design assets for the project.

| Folder | Contents |
|---|---|
| `pcb/` | KiCad project for the custom MCU board (schematic, layout, gerbers) |
| `schematics/` | Block diagrams, exported PDFs of schematics, sub-circuit notes |
| `enclosure/` | 3D-printed mounts, brackets, payload bay (STL / STEP / FreeCAD) |

See [`../docs/04-pcb-design.md`](../docs/04-pcb-design.md) for the board specification.
