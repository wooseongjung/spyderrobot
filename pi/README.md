# pi/

Software that runs on the Raspberry Pi 5.

| Folder | Contents |
|---|---|
| `telemetry/` | UART parser, CSV logger, web dashboard |
| `camera/` | Pi AI Camera capture scripts |
| `interface/` | Operator UI, joystick handler, PCA9685 servo control |

Architecture: [`../docs/06-pi-software.md`](../docs/06-pi-software.md)

Run on Pi OS Bookworm. Python 3.11+.
